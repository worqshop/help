Main Help

Main View

  • Tap the Import Project button to import GitHub projects into Worqshop.

Heroku Add-On

  • The add-on allows users to deploy projects to the Heroku cloud application platform.
  • The project that will be deployed must be a Heroku-deployable project.
  • Thus, for example, for Ruby projects, the Gemfile.lock file needs to be committed, please see https://devcenter.heroku.com/articles/bundler.
  • The push notification should be delivered in under 10 minutes. If it is not the case, please contact us.
  • Tap the Worqshop notifications in the iOS Notification Center to see the results of the Heroku deployment.

Runner Add-On

  • The add-on allows users to execute small Python or Ruby scripts on the Worqshop cloud.
  • It is useful to test small code functions directly from your device.
  • The script execution is limited to only fifteen seconds.
  • The add-on may need to provision additional servers for code execution. It may take up to 5 minutes.
  • Please contact us if it takes more than 5 minutes.
