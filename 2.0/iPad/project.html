Project View

  • The project view is the workspace for the selected repository or gist.
  • It has 3 view modes:
    • Showing the editor only.
    • Showing the editor and the assistant view.
    • Showing the editor and the navigation view.

Navigation View

  • The navigation view lists the files and issues for the selected repository.
  • It only lists the files for the selected gist.

Assistant View

  • The assistant view is an additional helper view.
  • It has 3 view modes:
    • Tabbed Editor.
    • Web Browser.
    • Automatic Reference.
  • Tap with two fingers on an item in the navigation view to open it in the assistant view editor.
  • The Automatic Reference displays the reference manual for the selected file based on the file type.

Editor

  • The editor saves the file automatically.
  • It provides syntax highlighting for Python, Ruby, HTML, CSS, JavaScript, C, Java, Objective-C, and PHP files.
  • A file cannot be opened at the same time in the main editor and in the assistant view editor.
