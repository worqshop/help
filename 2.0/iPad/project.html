Project Help

New Editor View

  • Use the editor view to edit files.
  • It has syntax highlighting support for Python, Ruby, HTML, CSS, and JavaScript files.
  • The editor autosaves file content during an editing session.

Navigator View

  • Use the navigator view to browse files or folders of the project.
  • The navigator view is on the left of the editor view.
  • It can be opened by swiping the editor view to the right.
  • Tap a folder and the view shows the content of the folder.
  • Tap a file and the content of the file will be shown in the editor view.

Library View

  • Use the library view to read the documentation for a particular file.
  • The library view is on the right of the editor view.
  • It can be opened by swiping the editor view to the left.
  • For Python files, it shows http://docs.python.org/.
  • For Ruby files, it shows http://www.ruby-doc.org/core.
  • For HTML files, it shows https://developer.mozilla.org/en/HTML/Element.
  • For CSS files, it shows https://developer.mozilla.org/en/JavaScript/Reference.
  • For JavaScript files, it shows https://developer.mozilla.org/en/CSS/CSS_Reference.
